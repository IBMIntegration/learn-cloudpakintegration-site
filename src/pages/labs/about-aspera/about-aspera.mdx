## About Aspera

- Aspera is a High Speed File Transfer Platform designed to move data of any size, over
  any IP based network at the fastest possible speeds.
- The name Aspera is an easy way of referring to a stack of components that all work in
  conjunction to bring together the Aspera ecosystem of software.
- The most basic way of representing the Aspera ecosystem of software is through the
  following stack"

  ![Aspera Stack](./images/stack.png)

There are a few key concepts this simple diagram helps explain:

1. FASP is the fundamental layer of all Aspera file transfers regardless of the way in
   which a transfer is initiated.
1. Aspera transfers are a 'Client/Server' architecture. At least one Server must be
   involved in a transfer.
1. There are many different ways of invoking FASP file transfers, each designed to
   accommodate a different type of workflow.
